# 🪙 Currency Converter | BRL Edition

Um conversor de moedas dinâmico e responsivo que utiliza APIs de cotação em tempo real para converter o Real (BRL) para as principais divisas globais e criptoativos.

---

## 🚀 Sobre o Projeto

Este projeto foi desenvolvido como parte do meu portfólio de **Desenvolvedor Front-end**. O objetivo principal foi criar uma ferramenta funcional que consome dados externos para fornecer cotações precisas de:

* 🇺🇸 **Dólar Americano (USD)**
* 🇪🇺 **Euro (EUR)**
* 🇬🇧 **Libra Esterlina (GBP)**
* ₿ **Bitcoin (BTC)**

### 💡 Diferenciais Técnicos
* **Consumo de API:** Utilização de `fetch` com `async/await` para garantir dados sempre atualizados.
* **Manipulação de DOM:** Atualização dinâmica dos elementos na tela sem recarregar a página.
* **Internacionalização:** Uso do objeto `Intl.NumberFormat` para formatação correta de moedas e símbolos.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído com uma stack moderna e limpa:

* **HTML5:** Estrutura semântica focada em acessibilidade.
* **CSS3:** Estilização personalizada (inspirada na paleta *Tokyo Night Storm*).
* **JavaScript (ES6+):** Lógica de conversão e integração com a API.
* **Node.js:** Ambiente de desenvolvimento e gerenciamento.
